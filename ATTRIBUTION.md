# ATTRIBUTION

This document details all external sources, tools, and assistance used in the creation of this project.

## Tools and Libraries
The following open-source Python libraries were used:

- `librosa` for audio loading, resampling, and mel-spectrogram generation
- `numpy` and `scikit-learn` for feature engineering, model training, hyperparameter tuning, and evaluation metrics
- `matplotlib` for visualization
- `pydub` (if installed) for optional audio conversions
- `google.colab` integration for GPU access and Drive mounting
- Standard Python libraries: `os`, `glob`, `warnings`, `sys`

These libraries were used according to their public documentation and licenses.

## Dataset Attribution
All audio clips used in this project were **collected, recorded, and labeled by me**. No copyrighted original material is included in this repository. The dataset was created through audio clips on my personal Google Drive storage.
